 
GrAPiC Graphics for Algo/Prog in C/C++ 
 
* Pour compiler
WINDOWS
  Compiler avec codeblocks : ouvrir le projet ....../Grapic/grapic.cbp
  Ou installer msys (http://www.mingw.org/wiki/msys)  puis make dans le terminal
LINUX
  Installer les packages SDL2 : sudo apt-get install libsdl2-dev libsdl2_ttf-dev libsdl2_image-dev
  puis make dans un terminal

Pour les developpeurs :
 hg clone https://forge.univ-lyon1.fr/hg/grapic

optionnel :
 Sous windows, 7-zip pour créer l'archive
 Soux windows, winscp pour faire la synchronisation avec la page web


* Les exemples
- main_demo.cpp: une micro démo ui contient un peu tout ce dont vous avez besoin pour faire un projet sympa
- main_particles.cpp: une démo de particules
